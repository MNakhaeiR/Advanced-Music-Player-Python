# Advanced Music Player

A simple advanced music player built with Python, PyQt5, and sounddevice.

## Features

- Play/Pause/Stop
- Next/Previous track
- Volume control
- Playlist management
- Load audio files

## How to Run

1. Install requirements:
   ```
   pip install -r requirements.txt
   ```
2. Run the app:
   ```
   python src/main.py
   ```
